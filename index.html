개인정보 처리방침

1. 개인정보 처리방침 개요

펭귄 알람(이하 "앱")은 사용자의 개인정보를 중요하게 생각하며, 개인정보보호법 등 관련 법령을 준수합니다. 본 개인정보 처리방침은 앱이 어떤 정보를 수집하고, 어떻게 사용하는지 설명합니다.

시행일: 2025년 1월 6일

2. 수집하는 정보

2.1 앱이 직접 수집하는 정보

펭귄 알람은 사용자의 개인정보를 직접 수집하지 않습니다.

앱에서 설정하는 알람 시간, 반복 요일, 알람 라벨 등의 정보는 모두 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

2.2 제3자 서비스를 통해 수집되는 정보

앱은 Google AdMob을 통해 광고를 표시합니다. AdMob은 다음 정보를 수집할 수 있습니다:

  • 광고 식별자 (Advertising ID)
  • 기기 정보 (기기 모델, 운영체제 버전)
  • IP 주소
  • 앱 사용 데이터

Google의 개인정보 처리방침: https://policies.google.com/privacy

3. 앱 권한

앱은 다음 권한을 사용하며, 각 권한은 명시된 목적으로만 사용됩니다:

권한 사용 목적
알람 및 리마인더 정확한 시간에 알람을 울리기 위해 필요
알림 알람 알림을 표시하기 위해 필요
진동 알람 시 진동 기능 제공
부팅 시 실행 기기 재시작 후 알람을 복원하기 위해 필요
카메라 QR 코드 스캔을 통한 알람 해제 기능 (선택)
오디오 파일 접근 사용자 지정 알람음 설정 (선택)

4. 정보의 저장 및 보안

모든 알람 데이터는 사용자의 기기 내부에만 저장됩니다:

  • 알람 정보는 기기의 로컬 데이터베이스에 저장
  • 설정 정보는 앱 내부 저장소에 저장
  • 외부 서버로의 데이터 전송 없음
  • 앱 삭제 시 모든 데이터가 기기에서 완전히 삭제됨

5. 아동의 개인정보

펭귄 알람은 만 13세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

6. 개인정보 처리방침 변경

본 개인정보 처리방침은 법령 변경이나 서비스 변경에 따라 수정될 수 있습니다. 변경 시 앱 내 공지 또는 본 페이지를 통해 안내드립니다.
